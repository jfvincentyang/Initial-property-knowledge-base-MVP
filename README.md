# 物业知识库 MVP

一个零依赖的最小可用原型，适合先验证“制度入库 + 基于知识库问答”这条主流程。

## 功能

- 录入物业规章制度文本
- 自动把制度切成知识片段
- 根据提问检索最相关的制度内容
- 返回结论、制度依据、建议步骤和命中片段
- 数据保存到本地 `data/knowledge-base.json`

## 运行

先执行测试：

```bash
npm test
```

再启动本地服务：

```bash
npm start
```

打开 [http://localhost:8000](http://localhost:8000)。

## 适合当前阶段的原因

这个版本没有接入大模型，也没有向量数据库，优点是：

- 部署简单
- 本地可直接演示
- 能先验证制度录入和问答流程

如果下一步要升级，可以继续加：

- Word / PDF 文件上传解析
- 更强的中文分词和检索
- 大模型总结答案
- 权限控制和操作日志
